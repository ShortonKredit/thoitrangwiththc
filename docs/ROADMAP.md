# Roadmap

## M0 — Foundation (included)

Data-driven catalog, wardrobe state, compatibility, locks, history, local save, procedural renderer, tests and deployment scripts.

## M1 — Local verification

Run Godot 4.7 import/test, fix any environment-specific issues, capture baseline screenshots and create the first Git commit.

## M2 — Art proof pack

Select the master female character anchor and create a small normalized PNG pack. Verify alignment before scaling content production.

## M3 — PNG renderer migration

Add body/hair/top/bottom/dress/shoe layers. Keep procedural fallback until the pack is complete.

## M4 — Web hardening

Browser PNG download, local save, fullscreen, loading performance, Chrome/Edge/Firefox checks and Netlify deployment.

## M5 — Face editor

Local file selection, decode/downscale, orientation, mask, drag/zoom/rotate, undo and privacy verification. No upload API.

## M6 — Polish

Audio toggle, accessibility labels, tablet layout, content expansion and final QA.
