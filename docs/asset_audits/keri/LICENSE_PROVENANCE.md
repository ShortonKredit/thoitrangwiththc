## License Text Found

`Credits.txt` in the Keri-C archive explicitly states `CC-BY-3.0` and requires credit to Konett. The template `README.md` states that Keri is `CC-BY license`, credits Konett as the original creator, and says the template/image modifications/interface/code were created by LunaLucid/Namastaii.

The template README also says credit to LunaLucid is not required for the code template or manipulated images, while credit to Konett is requested/required for Keri usage.

## Attribution Requirement

Minimum attribution for any later proof using this material should include:

- Konett as original Keri artist.
- LunaLucid/Namastaii as template/adaptation author when template-derived images or code structure are referenced.
- Source archive and source-page links retained in project documentation.
